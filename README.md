## Features

### Student Features
- View attendance by subject
- See attendance percentage
- List of recent lectures
- View assignments
- Chat with teachers

### Teacher Features
- Mark attendance for students
- Real-time attendance updates
- Upload assignments
- Chat with students

---

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend / Database:** Firebase (Authentication, Firestore, Hosting)
- **Real-time Communication:** Firebase Firestore, Chat system
